# Steps to run in local env

- [Install Rust](https://rust-lang.org/tools/install/)
- Clone this repo to your local - `git clone https://github.com/sivakarasala/mars_base.git`
- Run `mars_base_one` and `highscore_server` in two terminals.
- If not in mars_base directory do `cd mars_base`
- `cd mars_base_one`
- `cargo run`
- In another terminal
- `cd highscore_server`
- `cargo run`
